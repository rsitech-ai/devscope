# Governance

DevScope is currently a maintainer-led project.

## Roles and decisions

- Rafal Sikora is the copyright owner.
- RSI Tech is the public project, release, and security maintainer.
- `@s1korrrr` is the maintainer's operational GitHub account.
- Public and confidential project contact: `info@rsitech.ai`.
- Project website: <https://rsitech.ai>.
- Contributors propose changes through issues and pull requests.
- Maintainers decide by technical merit, product safety, compatibility, and long-term maintenance cost.
- Material architecture, security, privacy, or product-scope changes require an explicit maintainer decision recorded in the pull request or repository documentation.

No contributor has an automatic right to merge, publish, sign, notarize, or make
a release. Security fixes may be developed privately until coordinated disclosure.

## Releases

A release requires passing source, test, package, and review gates. Publishing
source does not imply that a downloadable binary is Developer ID signed,
notarized, App Store reviewed, or supported on untested hardware. Release notes
must state those boundaries accurately.

If the maintainer group grows, this document will be updated before granting
additional merge or release authority.
