# Support

DevScope is maintained on a best-effort basis without a support SLA.

RSI Tech maintains the public project at <https://rsitech.ai>. Public and
confidential project contact: `info@rsitech.ai`.

- Use [GitHub Issues](https://github.com/rsitech-ai/devscope/issues) for reproducible bugs and focused feature requests.
- Use [GitHub Discussions](https://github.com/rsitech-ai/devscope/discussions) for usage questions when Discussions are enabled.
- Follow [SECURITY.md](SECURITY.md) for private vulnerability reports.
- Read [PRIVACY.md](PRIVACY.md) before attaching logs, screenshots, exports, commands, or local paths.

When filing a bug, include the DevScope version or commit, macOS version, Mac
architecture, expected and actual behavior, and minimal reproduction steps.
Redact usernames, process arguments, project names, credentials, and private
paths. Do not ask contributors to run destructive commands against real work.

The full-feature product currently targets direct macOS distribution. Sandbox
validation does not imply feature parity with the direct-download build.
