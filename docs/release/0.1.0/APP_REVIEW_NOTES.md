# Draft App Review Notes

**Not submission-ready; owner and product approval required.**

DevScope is a local macOS process-inspection tool. The full product reads local process metadata, working directories, commands, and resource measurements, and can send TERM/KILL only after explicit confirmation with PID-birth validation and protected-service checks. It does not require an account or network backend.

The current Mac App Store sandbox build cannot inspect the local process set and therefore does not deliver the full product. Do not submit it as equivalent to the full build. If a reduced App Store edition is approved, these notes must be rewritten to describe its exact reduced functionality, how reviewers reach every feature, and why the listing/screenshots accurately represent that build.

Reviewer prerequisites for a future full-capability route: launch developer tools/services locally to populate rows. No demo account exists. No purchases or subscriptions exist. Permission guidance must explain that sandbox blocking is not fixable through System Settings.

Owner must supply review contact, support URL, privacy URL, content-rights/export answers, DSA status, age rating, accessibility declarations, and any regional restrictions.
